## 大陆网络说明

- 以上是 **GitHub Pages**。部分运营商/网络会慢或打不开。
- 若对方打不开交互页，改发同目录 PDF（微信/飞书本地文件，不依赖外网页面）：
  - `宠物主粮关键词需求分析报告.pdf`
  - `宠物食品及用品蓝海探测报告.pdf`
- 不建议用 jsDelivr / statically 的 HTML 链接：它们会把 HTML 当成纯文本，浏览器无法交互渲染。
